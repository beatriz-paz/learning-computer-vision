# Instrução

## Criar ambiente virtual

Dentro da pasta do projeto:

python3 -m venv vit-env


## Ativar o ambiente

source vit-env/bin/activate

## Instalar pacotes

pip install ipykernel
pip install tensorflow matplotlib scikit-learn opencv-python

## Conectar com o VS Code

Depois:

Ctrl + Shift + P

Python: Select Interpreter

Escolha:

vit-env/bin/python
